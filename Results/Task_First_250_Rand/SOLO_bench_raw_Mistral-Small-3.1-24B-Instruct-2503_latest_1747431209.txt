# The AI Content Generator

A simple Python program which use OpenAi's **gpt-3**, can generate text given the first few words of your sentence. You can find more about it [here](https://beta.openai.com/playground). I was really impressed by what I saw, and decided to write a simple wrapper script for the API so you could run this type or similar code very easily.

The code is very easy. A simple function which takes a starting prompt as an argument, and a **length** of words to generate - you control how long your story will be!

## Install Python Packages

You need Python3 installed on your system (I recommend using [Miniconda](https://docs.conda.io/en/latest/miniconda.html), but the latest 3.x version from python.org is fine).

To install all dependencies for this script
```
python -m pip install -r requirements.txt
```

## Generate text/Story

To generate a story with OpenAi, and save it to `output.txt`, pass the following arguments to [generate_text.py](generate_text.py):

`prompt`: A few words or sentences of the prompt you want the AI to start the story with.
```
Usage: python generate_text.py "I went for a walk in the forest." length=2000
```

For longer text/stories, increasing `lenght` parameter might be useful, but pay attention that each call may cost and will be added to your overall API usage.

To see all available options:
```
python generate_text.py --help
```

## Notes

I have included the `openai_api_key.env` file. You can replace this with your key, which you can [get on OpenAI](https://beta.openai.com/account/api-keys). For now, I will leave it as a blank (placeholder).
